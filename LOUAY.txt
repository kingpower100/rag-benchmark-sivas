## Project Overview

  This repository is a SIVAS ERP RAG benchmark project with two main systems:

  - Pipeline 1: RAG Execution
      - Loads SIVAS KB documents and questions.
      - Chunks documents, embeds chunks, builds/loads an index, retrieves context, orchestrates category-aware querying, generates answers, and writes run
        outputs.

  - Pipeline 2: Offline Evaluation
      - Reads Pipeline 1 results.jsonl.
      - Compares generated answers and retrieved evidence against qa_ground_truth_fixed.jsonl.
      - Produces per-question metrics, summaries, and benchmark reports.

  The current active benchmark dataset is the fixed SIVAS Dataset v1.0 set:

  data/raw/kb_documents_fixed.jsonl
  data/raw/questions_fixed.jsonl
  data/raw/qa_ground_truth_fixed.jsonl

  The codebase is organized around both Pipeline 1 and Pipeline 2, with active implementation under src/pipeline1/ and src/pipeline2/.

  ## Workflow Explanation

  SIVAS KB + Questions
        ↓
  Pipeline 1: RAG Execution
        - load KB and questions
        - chunk KB
        - embed chunks
        - build/load FAISS index
        - orchestrate category-aware query
        - retrieve chunks
        - generate answer with Ollama
        - write results.jsonl
        ↓
  Generated Answers + Retrieved Chunks
        ↓
  Pipeline 2: Offline Evaluation
        - load results.jsonl
        - load qa_ground_truth_fixed.jsonl
        - compute retrieval metrics
        - compute answer metrics
        - aggregate summaries
        ↓
  Metrics + Benchmark Report
